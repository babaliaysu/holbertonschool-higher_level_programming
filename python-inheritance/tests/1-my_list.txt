The ``1-my_list`` module
========================

Using ``MyList``
----------------
First, import the class:
    >>> MyList = __import__('1-my_list').MyList

Create an instance and add elements:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test the print_sorted method:
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Verify that the original list is still unsorted:
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test with an empty list:
    >>> empty_list = MyList()
    >>> empty_list.print_sorted()
    []

Test with already sorted list:
    >>> sorted_list = MyList([1, 2, 3])
    >>> sorted_list.print_sorted()
    [1, 2, 3]
